## About This Repository

This repository contains all practical programs implemented as part of the Object Oriented Programming (OOP) laboratory coursework.

The programs are written in Java and organized practical-wise according to the official syllabus. Each practical demonstrates specific concepts of Java and Object-Oriented Programming.

---

## Topics Covered

- Java Basics
- Classes and Objects
- Method Overloading
- Static Members and Blocks
- Inheritance and Polymorphism
- Interfaces and Abstract Classes
- Packages and Access Specifiers
- Exception Handling
- Multithreading
- File Handling
- Generics and Collections
- JavaFX

---

## Repository Structure

Each practical is organized into separate folders based on topic sets.  
All source files are written in Java and can be compiled and executed individually.
